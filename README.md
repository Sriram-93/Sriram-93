## 🌟 Featured Project

### ⚡ FlashcardAI — [View Repository](https://github.com/Sriram-93/FlashcardAI)

> An AI-powered study platform that converts any PDF into smart flashcards, quizzes, and voice-guided study sessions.

| Feature | Details |
|---|---|
| 🧠 AI Generation | Groq LLM extracts questions, answers & difficulty levels from PDFs |
| 🔍 Vector Search | HuggingFace embeddings + MongoDB Atlas for semantic retrieval |
| 🗣️ Voice Mode | Text-to-speech for audio-guided study sessions |
| 📊 Progress Tracking | Mastery tracking across Easy, Medium & Hard levels |
| 🔐 Auth System | JWT-based secure signup & login |
| 🚀 Deployed | Live on Vercel |

**Stack:** `React` · `Node.js` · `Express` · `MongoDB` · `Groq` · `LangChain` · `HuggingFace`
